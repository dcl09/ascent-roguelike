# LDTS Project <Class 2> <Group 6> – Ascent


In this roguelike dungeon-crawler, you, the player, must go through several dangerous levels of the deep-dark caves below Narnia, in which you will fight through hordes of enemies to ascend to the next level. As you ascend, the danger will continuously escalate, but so will your ability to deal with it.
